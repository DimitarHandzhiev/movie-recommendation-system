# Movie Recommendation System

A hybrid movie recommendation system built with Python and LightFM using the MovieLens dataset.

## Goal
The goal of this project is to build a recommendation engine that combines collaborative filtering and content-based signals.

## Dataset
- MovieLens ml-latest-small

## Tech stack
- Python
- pandas
- numpy
- LightFM
- scikit-learn

## Planned improvements
- Item features from movie genres
- Top-N recommendation generation
- Evaluation with train/test split
- Optional Streamlit interface
